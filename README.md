# CRM Ассистент

Простая CRM-система с AI-ассистентом на базе Groq API. Позволяет управлять контактами, сделками и задачами через веб-интерфейс или через чат с ИИ.

## Стек

- **Backend:** Node.js, Express
- **База данных:** SQLite (better-sqlite3)
- **AI:** Groq API, модель `meta-llama/llama-4-scout-17b-16e-instruct`
- **Frontend:** Vanilla HTML/CSS/JS

## Возможности

- Управление контактами, сделками и задачами (CRUD)
- AI-ассистент с function calling — реально создаёт, изменяет и удаляет записи в БД по команде пользователя
- История диалога с ассистентом
- Серверная валидация данных из сообщений пользователя

## Установка

```bash
npm install
```

Создать файл `.env`:

```
GROQ_API_KEY=ваш_ключ
PORT=3000
```

## Запуск

```bash
node server.js
```

Открыть `index.html` в браузере.

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | /contacts | Список контактов |
| POST | /contacts | Создать контакт |
| PUT | /contacts/:id | Обновить контакт |
| DELETE | /contacts/:id | Удалить контакт |
| GET | /deals | Список сделок |
| POST | /deals | Создать сделку |
| PUT | /deals/:id | Обновить сделку |
| DELETE | /deals/:id | Удалить сделку |
| GET | /tasks | Список задач |
| POST | /tasks | Создать задачу |
| PUT | /tasks/:id | Обновить задачу |
| DELETE | /tasks/:id | Удалить задачу |
| POST | /chat | Отправить сообщение ассистенту |
| GET | /messages | История сообщений |
